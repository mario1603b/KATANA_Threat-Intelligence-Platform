# ⚔️ KATANA - Threat Intelligence Analyzer

![Python](https://img.shields.io/badge/python-3.x-blue.svg)
![CustomTkinter](https://img.shields.io/badge/GUI-CustomTkinter-darkgreen.svg)
![License](https://img.shields.io/badge/license-MIT-green)

**KATANA (Pro Edition)** is a modern, lightweight desktop application designed for Security Operations Center (SOC) analysts and network administrators. It automates the forensic analysis of logs exported from **Sophos Firewalls**, rapidly identifying brute-force attacks, targeted accounts, and the geographical origin of threat actors.

## 🚀 Features

* **Universal Log Parsing:** Uses advanced regex to extract IP addresses and targeted usernames from raw Sophos CSV logs, regardless of the specific export format (VPN Portal, Admin, System, etc.).
* **Batch Geolocation:** Integrates with the `ip-api.com` service using batch requests to geolocate thousands of attacking IPs in seconds without hitting rate limits.
* **Modern macOS-Style GUI:** Built with `customtkinter`, featuring a sleek, responsive interface, native Dark/Light mode toggling, and fluid transitions.
* **Threat Dashboards:** * 📊 **Geography:** Matplotlib-powered bar charts displaying the top attacking countries.
  * 👤 **Compromised Users:** Horizontal bar charts identifying which exact accounts (e.g., `administrator`, `vpnuser`) are being targeted by dictionaries.
  * 🌍 **Global Threat Map:** Generates an interactive, high-resolution 3D/2D topological heat map using `plotly`, which opens securely in your default web browser.
* **Automated Reporting:** Silently exports a clean, enriched CSV file (`Resultado_KATANA.csv`) containing timestamps, IPs, countries, and usernames, ready to be attached to incident response tickets.

## 🛠️ Prerequisites & Installation

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/yourusername/katana-threat-analyzer.git](https://github.com/yourusername/katana-threat-analyzer.git)
   cd katana-threat-analyzer
   ```
Create a virtual environment (Recommended):

```Bash
python -m venv venv
source venv/bin/activate  # On Windows use: venv\Scripts\activate
```
Install dependencies:
Ensure you have the required libraries installed:

```Bash
pip install customtkinter pandas requests matplotlib plotly Pillow
```
🖥️ Usage
Export Logs: Access your Sophos Firewall WebAdmin -> Log Viewer. Filter your desired view (e.g., VPN Portal Authentication) and export the records as a CSV file.

Run KATANA:

```Bash
python katana.py
```
Analyze: * Click "📂 Select Log" and choose your exported CSV.

Click "⚡ Start Analysis".

Browse through the tabs on the right panel to view the generated intelligence and launch the interactive web map.

📦 Building the Executable (Windows)
If you want to distribute KATANA to your team without requiring them to install Python, you can compile it into a portable Windows executable using PyInstaller.

Install PyInstaller:

```Bash
pip install pyinstaller
```
Build the app (Using --onedir is highly recommended over --onefile for faster startup times due to the heavy data science libraries):

```Bash
pyinstaller --noconfirm --onedir --windowed --name "KATANA_v2.0" katana.py
```
Locate your compiled tool inside the generated dist/ folder.

⚠️ Disclaimer
This tool uses the free endpoint of ip-api.com for geolocation. Please be mindful of their Terms of Service and rate limits (which KATANA handles automatically via batching). This software is provided "as is" for defensive and forensic purposes only.
